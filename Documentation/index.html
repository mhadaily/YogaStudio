File Structure

            - assets/css     (Template CSS)
            - assets/ico     (Favicon and apple touch icon)
            - assets/img     (All Images)
            - assets/js      (Template JS)
            - assets/fonts   (Yoga fonts files)
            - assets/plugins (All external libs. We keep all of them in this folder to make updates easily.)

 

HTML Structure

The template is based on Bootstrap Framework - http://getbootstrap.com/
The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport.
Below 480px viewports, the columns become fluid and stack vertically.

Create a .row and add the appropriate number of .col-sm-* columns.

HTML Markup

        <div class="row">
            <div class="col-sm-12">
                Level 1 column
                <div class="row">
                    <div class="col-sm-6">Level 2</div>
                    <div class="col-sm-6">Level 2</div>
                </div>
            </div>
        </div>

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

To change the embedded font, please take a look in the head part of assets/css/theme.css:

            @import url('http://fonts.googleapis.com/css?family=Raleway:400,100,200,300,500,600,700,800,900');

To change the font, first go to http://www.google.com/webfonts choose a font and add the generated code to the top of theme.css.

Then edit font-family in 

            body {
                font-family: 'Raleway', sans-serif;
                font-size: 18px;
                line-height: 1.33;
            }

Schedule

Schedule tabs created with bootstrap tabs and you can find more documentation here http://getbootstrap.com/javascript/#tabs

Just some notes for easy understanding:

  • <div class="schedule-wrapper clear"> - wrap all schedule content
  • <div class="schedule-tabs main"> - "main" for days (Monday, Tuesday, ... ) and each tab has id 'tab-first', 'tab-second' ...
  • <div class="tab-content main"> - display content for tabs of first level and each tab-pane has correct id 'tab-first', 'tab-second' as tab switcher

PHP Scripts

Registration & Contact Forms

For Registration & Contact Form to work only thing you need to do is to setup your email address and your name.
To do this, please open "register.php & contact.php" file and change these lines:

        $to      = 'you@yourdomain.com';
        $subject = 'Your Subject';

One notice: this will be functional only on your server, php files can not be executed on your local computer.
